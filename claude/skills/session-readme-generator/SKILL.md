# Purpose

Transform a learning session into a structured knowledge document that can be stored in a Git repository, Obsidian vault, Confluence page, or personal notes system.

The goal is knowledge retention and future revision.

# When To Use

Use this skill when:

* User finishes a learning session
* User asks for notes
* User asks for a README
* User asks for a study guide
* User asks for a learning summary
* User wants documentation of the current conversation

# Output Requirements

Generate Markdown only.

The document must be self-contained and understandable without reading the original chat.

# Document Structure

# Title

A concise title representing the topic.

## Overview

Brief explanation of the topic.

## Table of Contents

Generate links to all major sections.

## Key Concepts

For each concept:

### Concept Name

Definition.

Why it matters.

How it relates to other concepts.

Example.

## Mental Models

Explain useful analogies discussed during the session.

## Examples Covered

Document every example explored during the conversation.

For each example:

* Problem
* Explanation
* Solution
* Key takeaway

## Important Relationships

Show how concepts connect.

Example:

Neural Network
→ Weighted Sum
→ Activation Function
→ Loss Function
→ Backpropagation
→ Gradient Descent

## Formulas

List all formulas discussed.

For each formula:

* Formula
* Variable meanings
* Practical interpretation

## Common Mistakes

Document misconceptions corrected during the discussion.

## Revision Notes

Short memory-friendly bullet points.

## Glossary

Definitions of important terms.

## Next Topics To Learn

Generate logical follow-up topics.

# Writing Rules

* Preserve technical accuracy.
* Do not omit important explanations.
* Include examples whenever available.
* Expand shorthand explanations into complete notes.
* Prefer educational clarity over brevity.
* Write as if creating documentation for future study.
* Make the output suitable for direct storage as README.md.

## Big Picture

Explain:

- What problem does this topic solve?
- Why was it invented?
- Where does it fit in the larger system?
- What comes before it?
- What comes after it?
